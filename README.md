# coby

An application to manipulate images on the web 

# HOW TO INSTALL 

# If using PIPENV 

`pipenv shell`

`python app.py`
 
### NOTE: if on Mac OS, use following command


`pipenv shell`

 `python3 app.py`
 
 ### Server will start listening by default on port 5000, can be changed with config variable
